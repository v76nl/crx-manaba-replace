# インストール手順
以下の手順で manaba Replace を Google Chrome や Microsoft Edge にインストールしてください。
  

## 1. ファイルをダウンロードする
1. このページ右上付近、緑色の「Code」ボタンをクリックします。
3. 表示されたメニューの中から「Download ZIP」をクリックします。  
4. ZIPファイルがダウンロードされます。  


## 2. ZIP ファイルを解凍 (展開) する
### Windows の場合
1. ダウンロードしたZIPファイルを探します。  
   通常は「ダウンロード」フォルダに保存されています。
2. ZIPファイルを右クリックし、「すべて展開」を選択します。
3. 展開先フォルダを指定して解凍します。  
   - 例：`C:\Users\あなたのユーザー名\Documents\ChromeExtensions\manaba-replace`
   - 「あなたのドキュメントフォルダ内の、 `ChromeExtensions` フォルダの中に、 `manaba-replace` フォルダを置く」という意味です。
   - 注意：  
        拡張機能フォルダは、あとからも使うため、 「ダウンロード」フォルダやデスクトップに置きっぱなしにしないことをおすすめします。 `Documents` (「ドキュメント」) 内などに保存してください。
4. 解凍が完了したら、`manaba-replace` フォルダの場所を覚えておきます。

### macOS の場合
1. ダウンロードしたZIPファイルを探します。  
   通常は「ダウンロード」フォルダに保存されています。
2. ZIPファイルをダブルクリックすると、自動的に解凍されます。
3. 解凍されてできたフォルダを移動します。  
   - 例：`/Users/あなたのユーザー名/Documents/ChromeExtensions/manaba-replace`
   - 「あなたのドキュメントフォルダ内の `ChromeExtensions` フォルダの中に `manaba-replace` フォルダを置く」という意味です。
   - 注意：  
        拡張機能フォルダは、あとからも使うため、 「ダウンロード」フォルダやデスクトップに置きっぱなしにしないことをおすすめします。 `Documents` (「ドキュメント」) 内などに保存してください。
4. 移動後の `manaba-replace` フォルダの場所を覚えておきます。


## 3. Chrome の拡張機能ページを開く
- 注：Microsoft Edgeを使っている場合は、「Chrome」を「Edge」に読み替えてください。
1. Google Chrome を起動します。
2. アドレスバーに `chrome://extensions/` と入力して、Enter キーを押します。
3. 拡張機能ページが表示されます。


## 4. デベロッパーモードを有効にする
1. 拡張機能ページの右上にある「デベロッパーモード」 (または「開発者モード」) というボタンを探します。
2. ボタンをオンにします。  
   - オンにすると、左上に「パッケージ化されていない拡張機能を読み込む」のボタンが表示されます。


## 5. 解凍したフォルダを読み込む
1. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。  
2. フォルダを選択するダイアログが開きます。
3. 先ほど解凍した先のフォルダを選択します。  
   - Windows の例：  
     `C:\Users\あなたのユーザー名\Documents\ChromeExtensions\manaba-replace`
   - macOS の例：  
     `/Users/あなたのユーザー名/Documents/ChromeExtensions/manaba-replace`
4. フォルダを選択して「フォルダーの選択」をクリックします。


## 6. インストール完了を確認する
1. 読み込みが成功すると、拡張機能ページの一覧にmanaba Replaceが表示されます。
2. 拡張機能の右下あたりにあるスイッチがオンになっていることを確認してください。


## 7. 変換する文字列の設定
1. Chrome のツールバー (右上のパズルアイコン) をクリックし、出てきた manaba Replace をクリックします。するとウィンドウが現れます。
2. 現れたウィンドウに `変えたい文字列=>代わりに表示したい文字列` の形式で入力してください。
    - `=>` は半角であることに注意してください。分からない場合はコピー&ペーストしてください。
3. manabaを再読み込みすると、変換が行われるようになります。
